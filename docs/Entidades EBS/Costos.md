## ¿Qué es?
Registro de todos los gastos reales incurridos durante un [[Evento]] o [[Proyecto]].

> La contabilidad formal vive en **Odoo**.
> Este sistema registra costos operativos para dar visibilidad al PM en tiempo real.

## Tipos de costo operativo
| Tipo | Origen |
|---|---|
| Renta de recurso externo | [[Renta]] confirmada |
| Personal eventual | [[Turno]]s de destajo |
| Viáticos | [[Viaticos]] del personal foráneo |
| Hospedaje | [[Hospedaje]] del personal |
| Vuelos | [[Vuelos]] del personal |
| Transporte extra | [[Vehículo]]s rentados |
| Compras | Materiales consumibles |

## Presupuesto vs Real
- [[Presupuesto]] = cuánto se puede gastar (estimado)
- Costos = cuánto se gastó realmente
- Varianza = diferencia entre ambos

## Relaciones
- Cada costo está asociado a un [[Evento]]
- Visibles para [[Project Manager]] y [[Gerente Operaciones]]
- Facturación final ocurre en Odoo
