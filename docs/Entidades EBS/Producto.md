## ¿Qué es?
Referencia comercial de algo que EBS vende o cotiza.

> Los Productos viven en **Odoo**. Este sistema no los gestiona.

## Diferencia clave con Recurso
| Producto | Recurso |
|---|---|
| Referencia comercial (qué se vende) | Instancia física (qué se usa) |
| Vive en Odoo | Vive en este sistema |
| Tiene precio de venta | Tiene disponibilidad y ubicación |
| "Servicio Roof System 12x8" | Las 48 piezas específicas que lo arman |

## Por qué existe esta nota
Cuando alguien dice "Producto" en contexto de cotización → Odoo.
Cuando dice "Recurso" en contexto de disponibilidad → este sistema.

El [[Despiece]] traduce Productos en Recursos específicos.
