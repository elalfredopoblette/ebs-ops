## ¿Qué es?
Representación técnica y visual de cómo será la estructura
de un [[Evento]]: dimensiones, distribución, accesos, puntos de anclaje.

## Tipos de plano en EBS
| Tipo | Contenido |
|---|---|
| Plano estructural | Detalles técnicos del [[Categoria de Servicios/STEEL\|STEEL]]: alturas, cargas, conexiones |
| Plano de distribución | Layout del [[Venue]]: dónde va cada elemento EPS |
| Plano de accesos | Rutas de vehículos, zonas de descarga, restricciones |
| Plano de obra civil | Dimensiones del recinto, coordenadas de referencia |

## Origen
El área de [[Diseño]] genera los planos a partir del [[Brief]] y las
restricciones del [[Venue]].

## Usos operativos
- El [[Jefe de Steel]] usa el plano estructural en campo
- El [[Supervisor EPS]] usa el plano de distribución
- La [[Logistica de Carga]] usa el plano de accesos

## Relación con el Despiece
El [[Despiece]] lista los recursos. El Plano muestra cómo ensamblarse.
Ambos son necesarios para la ejecución.

## Versionamiento
Si el [[Venue]] tiene restricciones imprevistas o el cliente modifica el alcance,
el plano se actualiza. Toda modificación al plano puede implicar cambios al [[Despiece]].

## Relaciones
- Generado por [[Diseño]]
- Entregado al [[Project Manager]] junto con el [[Despiece]]
- Referencia durante [[Load In]], [[ShowDay]] y [[Load Out]]
