## ¿Qué es?
Responsable de toda la operación de EBS: planeación de recursos,
capacidad operativa, calidad de ejecución y control de costos operativos.

## Responsabilidades
| Ámbito | Responsabilidad |
|---|---|
| Capacidad | Decide si EBS puede asumir un nuevo [[Evento]] |
| Recursos | Aprueba [[Renta]]s y compras de alto valor |
| Conflictos | Resuelve [[Conflicto]]s que el PM no puede resolver |
| Personal | Asignación de [[Project Manager]]s a eventos |
| Costos | Controla [[Costos]] operativos vs [[Presupuesto]] |
| Proyección | Supervisa [[Proyección]] de demanda futura |

## Autoridades
- Única persona que puede aprobar Rentas sobre cierto umbral
- Puede reasignar PMs entre eventos
- Cierra eventos que el PM no puede cerrar
- Acceso completo a [[Inventario]] y disponibilidad global

## Relaciones
- Supervisa a todos los [[Project Manager]]s
- Coordina con [[Gerente Comercial]] sobre compromisos al cliente
- Reporta a [[Dirección]]
